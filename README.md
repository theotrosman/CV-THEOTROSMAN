## About

This project is my personal interactive curriculum / portfolio website.  
Instead of using a traditional static PDF, I wanted to create a developer-focused experience that feels modern, dynamic and memorable.

The goal was to combine:
- clean visual design
- smooth interactions
- strong frontend architecture
- responsive layouts
- a unique personal identity

---

## Features

- Modern responsive UI
- Smooth animations & transitions
- Interactive sections
- Clean and minimal aesthetic
- Optimized performance
- Fully customizable
- Mobile friendly
- Fast loading experience

---

## Tech Stack

- React
- TypeScript
- TailwindCSS
- Framer Motion
- Vite
- Node.js

---

## Philosophy

Most CVs look the same.

I wanted something that feels closer to a real product:
a personal experience that reflects how I design and build software.

This project focuses on:
- attention to detail
- visual hierarchy
- UX
- frontend engineering
- identity through design
